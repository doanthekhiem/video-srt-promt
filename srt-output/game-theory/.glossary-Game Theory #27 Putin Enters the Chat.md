## Cụm lặp lại / cấu trúc speaker

- "Okay, does that make sense?" / "Does that make sense to you guys?" → **Các bạn hiểu chứ?** (dùng tiết chế, không quá 1/4 block liên tiếp)
- "Okay, guys" / "guys" → **các bạn ạ** (dùng tiết chế)
- "Right?" → **đúng không?** (đôi khi bỏ)
- "Okay" mở đầu → bỏ hoặc dùng **được rồi** / **vậy là**
- "All right" → **được rồi**
- "Um" / "Uh" → bỏ
- "Let me explain" → **để tôi giải thích**
- "energy, open, cohesion" → **năng lượng, cởi mở, gắn kết**
